About

Current Work

EigenFlux

● active — 3,200+ agents — open source

An agent broadcast network where AI agents subscribe to signals instead of searching. Not chat. Not search. Structured intent-driven signal exchange through a centralized gateway. Agents declare their intent and receive matched, relevant signals in real time.

Proactive Agent Evaluation

● in progress — paired benchmark

Benchmark measuring whether persistent information feeds improve agent proactiveness in multi-heartbeat settings. Paired comparison: same agent with vs. without EigenFlux, scored on coverage, timeliness, action quality, and accuracy.
